# PARSHVA-TECHNOLOGIES

1. Framework & Routing
TanStack Start: A modern full-stack React framework for building server-rendered (SSR) applications.
TanStack Router: A fully type-safe routing engine for React.
TanStack React Query: Handlers for server state, caching, data fetching, and mutations.
Nitro: Serves as the deployment/SSR engine (configured with the Cloudflare Pages/Module preset).
2. Languages & Runtime
React 19: Using the latest React core features.
TypeScript: Strict type-checking throughout the project.
Vite: The build tool and development server.
Bun/npm: Package lockfiles are available for both runtimes.
3. Styling & Animations
Tailwind CSS v4: The utility-first CSS framework integrated via Vite.
Framer Motion: Handles premium page transitions, scroll triggers, and complex micro-animations.
Lenis: Provides smooth momentum scrolling on the client side.
Lucide React: Vector icons.
4. Interactive 3D Graphics
Three.js / React Three Fiber / @react-three/drei: Powering the custom interactive elements like the landing page background canvas and globe.
5. Backend & Forms
Supabase Client: Standard integration for database access, authentication, and backend functions.
React Hook Form & Zod: Validated schema-based form management.
